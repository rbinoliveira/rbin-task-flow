# Taskmaster Commands

Use the Taskmaster MCP server to manage project tasks and workflows.

## Available Commands

- `task-master init` - Initialize Taskmaster in the project
- `task-master models --setup` - Configure AI models (Claude as default)
- `task-master create` - Create a new task
- `task-master list` - List all tasks
- `task-master status` - Check task status
- `task-master brief` - Generate task brief
- `task-master complete` - Mark task as complete

Follow the development workflow defined in `.cursor/rules/taskmaster/dev_workflow.mdc`
